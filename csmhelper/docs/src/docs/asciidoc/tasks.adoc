=== Внутренние задачи

В сервисе `csmhelper` внутренние периодические задачи (`BackgroundService`, `IHostedService`, `Hangfire`, `Quartz`) не используются. Все операции выполняются синхронно в рамках обработки HTTP-запроса.

Для задач, инициируемых пользователем и состоящих из нескольких параллельных вызовов Jira (например, `POST /Cleanup/ProcessTasks`), применяется in-process параллелизм через `Task.WhenAll` с батчингом по `3` задачи и паузой `500 ms` между батчами — см. соответствующий метод.
