==== Конфигурация приложения

Источники конфигурации — `appsettings.json` и `appsettings.Development.json`.

[cols="1,1,2"]
|===
|Ключ |Значение по умолчанию |Описание

|`Logging:LogLevel:Default`
|`Debug`
|Уровень логирования по умолчанию

|`Logging:LogLevel:Microsoft`
|`Warning`
|Уровень для пространства имён `Microsoft.*`

|`Logging:LogLevel:Microsoft.Hosting.Lifetime`
|`Information`
|Уровень для логов запуска/остановки приложения

|`Logging:LogLevel:csmhelper`
|`Debug`
|Уровень для собственных классов сервиса

|`AllowedHosts`
|`*`
|Список разрешённых Host-заголовков

|`Jira:BaseUrl`
|`https://jira.moscow.alfaintra.net`
|Базовый URL Jira. На текущий момент *не* читается из конфигурации в коде — `JiraService` и `GantService` хардкодят URL `https://jira.moscow.alfaintra.net`. Параметр зарезервирован

|===

==== Параметры HTTP-клиента к Jira

Регистрируется в `Program.cs` через `AddHttpClient<IJiraService, JiraService>`:

[cols="1,1,2"]
|===
|Параметр |Значение |Описание

|`BaseAddress`
|`https://jira.moscow.alfaintra.net`
|Базовый URL Jira

|`Timeout`
|`30 секунд`
|Таймаут запроса (для `JiraService`). В `GantService` создаётся отдельный `HttpClient` с таймаутом `120 секунд`

|`ServerCertificateCustomValidationCallback`
|кастомный
|Доверяет сертификату, если: ошибок валидации нет, либо `cert.Issuer` содержит `alfaintra`, либо ошибка только `RemoteCertificateChainErrors`

|===

В `GantService.FetchJiraTasksAsync` используется отдельный одноразовый `HttpClient`. Если `request.VerifySsl == false`, валидация SSL отключается полностью (`(_, _, _, _) => true`).

==== Параметры планирования (Gant)

В `GantScheduleSettings`:

[cols="1,1,2"]
|===
|Параметр |Значение |Описание

|`TaskTransitionLagMinutes`
|`60`
|Лаг (минуты) между концом блокирующей задачи и стартом блокируемой

|`SpBase`
|`21`
|Базовое число SP в спринте (приватная константа)

|`SprintHoursTotal`
|`80`
|Часов в спринте на одного сотрудника (приватная константа)

|`AgilePercent`
|`15`
|Процент времени, забронированный под agile-активности и не участвующий в `WorkHours` (приватная константа)

|`SpToWorkHours`
|`SprintHoursTotal * (1 - AgilePercent / 100) / SpBase ≈ 3.238`
|Сколько чистых рабочих часов в одном SP

|===
