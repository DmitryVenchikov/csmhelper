=== `GET /Gant` Страница построения диаграммы Ганта


==== Назначение
Защищённая страница, отдающая UI диаграммы Ганта: пользователь задаёт список проектов Jira и состав сотрудников (имя, роль, рабочие часы, обед, рабочее окно), после чего фронтенд вызывает `POST /Gant/Generate`. На странице отображаются графики и таблицы по результатам генерации.

Маршрут регистрируется через атрибуты `[Route("Gant")]`, `[HttpGet("")]`, `[HttpGet("Index")]` — то есть отвечает и на `/Gant`, и на `/Gant/Index`.

==== Алгоритм работы
. Система получает запрос `GET /Gant` (или `GET /Gant/Index`)
. Система проверяет `Session["IsAuthenticated"] == "true"`
.. Если флаг отсутствует — Система пишет `LogWarning("Попытка доступа к Gant без авторизации")` и выполняет редирект `302` на `/Auth/Login?returnUrl=/Gant`
. Система рендерит представление `Views/Gant/Index.cshtml`
. Система возвращает HTML с HTTP-кодом = 200 OK
