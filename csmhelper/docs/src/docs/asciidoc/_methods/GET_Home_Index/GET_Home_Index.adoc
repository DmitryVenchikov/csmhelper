=== `GET /` Главная страница приложения


==== Назначение
Корневой маршрут приложения (route default — `Home/Index`). Отдаёт публичную лендинг-страницу `Views/Home/Index.cshtml` со списком модулей: `Jira` (создание связанных задач), `Cleanup` (массовая обработка), `Gant` (диаграмма Ганта). Никакой бизнес-логики не выполняет.

==== Алгоритм работы
. Система получает запрос `GET /` (или `GET /Home/Index`)
. Система рендерит представление `Views/Home/Index.cshtml`
. Система возвращает HTML c HTTP-кодом = 200 OK
