=== `GET /Auth/Login` Отображение формы входа


==== Назначение
Метод предназначен для отображения страницы входа в Jira. Является публичным (не требует уже выполненной аутентификации).

Принимает опциональный query-параметр `returnUrl`, в который пользователя вернут после успешного логина. Если параметр не передан, по умолчанию используется `/Jira`. Значение прокидывается в представление через `ViewData["ReturnUrl"]`.

==== Алгоритм работы
. Система получает запрос `GET /Auth/Login` с опциональным query-параметром `returnUrl`
. Если `returnUrl` пустой/`null` — Система устанавливает `returnUrl = "/Jira"`
. Система кладёт `returnUrl` в `ViewData["ReturnUrl"]`
. Система пишет в `Console` сообщение `Login page requested with returnUrl: <returnUrl>`
. Система возвращает представление `Views/Auth/Login.cshtml` с HTTP-кодом = 200 OK
