=== `GET /Cleanup` Страница массовой обработки задач


==== Назначение
Защищённая страница, отдающая UI массового поиска и обработки задач Jira: на ней пользователь задаёт фильтры (проекты, статусы, эпики, спринты, даты создания/обновления) и решает, что сделать с найденными задачами — удалить или перевести в выбранный статус.

==== Алгоритм работы
. Система получает запрос `GET /Cleanup`
. Система проверяет локальный флаг `Session["IsAuthenticated"] == "true"`
.. Если флаг отсутствует — Система выполняет редирект `302` на `/Auth/Login?returnUrl=/Cleanup`
. Система вызывает `IJiraService.IsAuthenticatedAsync()`
.. Если сервис вернул `false` — Система выполняет редирект `302` на `/Auth/Login?returnUrl=/Cleanup`
. Система рендерит представление `Views/Cleanup/Index.cshtml` (с `ViewBag.IsAuthenticated = true`)
. Система возвращает HTML с HTTP-кодом = 200 OK
